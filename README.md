# 🎯 Missão da Semana #3: Protocolo de Economia 💰

Este projeto foi desenvolvido como parte de um desafio prático de **Fullstack** proposto pela **Codi Academy**. O objetivo principal da missão foi criar uma **Barra de Progresso Interativa** para monitoramento de metas financeiras, unindo lógica matemática e manipulação visual avançada.

---

## 📝 O Desafio

A missão consistia em criar uma ferramenta de visualização de metas de economia com regras de feedback visual em tempo real:
1. O sistema recebe a **Meta de Economia** e o **Valor Economizado**.
2. O sistema calcula a porcentagem de progresso (**Economizado / Meta * 100**).
3. O sistema deve fornecer uma resposta visual dinâmica:
    * **Progresso < 50%**: Barra amarela (Inicializando Carga).
    * **Progresso >= 50%**: Barra verde (Carga em Progresso).
    * **Progresso = 100%**: Efeito **Overcharge** (Brilho e pulsação neon).

---

## 📂 Estrutura e Organização

Para manter o código limpo e profissional, o projeto foi estruturado em três camadas:

* **\`index.html\`**: Estrutura do painel de controle e container do núcleo de energia.
* **\`style.css\`**: Design futurista focado em **Cyberpunk Aesthetics**, com gradientes neon e efeitos de desfoque.
* **\`script.js\`**: Lógica responsável pelo cálculo da porcentagem e controle de opacidade do núcleo.

---

## ✨ Funcionalidades e Bônus

Fui além dos requisitos básicos para entregar uma interface de nível "Sistema Operacional":
- [x] **Núcleo de Energia (Core)**: Barra de progresso com brilho dinâmico na ponta.
- [x] **Sistema de Opacidade**: O bloco de luz permanece oculto até que uma carga seja detectada.
- [x] **Animação Elástica**: Transição suave utilizando \`cubic-bezier\` para um efeito premium.
- [x] **Modo Overcharge**: Animação de pulsação luminosa ao atingir 100% da meta.
- [x] **Interface Sci-Fi**: Fundo em gradiente radial e tipografia mono-espaçada.

---

## 🛠️ Tecnologias

* **HTML5** (Semântica de formulários)
* **CSS3** (Animations, Keyframes, Backdrop-filter, Pseudo-elementos)
* **JavaScript** (Manipulação de classes, estilos dinâmicos e lógica de validação)

---

## 🚀 Como Visualizar

1. Clone o repositório.
2. Certifique-se de que o \`script.js\` e o \`style.css\` estejam na mesma pasta que o HTML.
3. Abra o \`index.html\` no seu navegador preferido.
